# md+ — правила разметки (Таблица 1)

## Inline-форматирование

### Синтаксис

| Конструкция | Результат | Пример |
|------------|-----------|--------|
| `//текст//` | Курсив | `//привет//` → *привет* |
| `**текст**` | Жирный | `**важно**` → **важно** |
| `__текст__` | Подчёркнутый | `__подч__` → подчёркнуто |
| `''текст''` | Верхний индекс | `''2''` → ² |
| `,,текст,,` | Нижний индекс | `,,2,,` → ₂ |
| `~~текст~~` | Зачёркнутый | `~~удалено~~` → ~~удалено~~ |
| `==текст==` | Подсветка | `==важно==` |
| `!!текст!!` | Спойлер | `!!спойлер:текст!!` |
| `!!!текст!!!` | Блок спойлер | `!!!спойлер!!!` |
| `++текст++` | Вставка (insert) | `++новое++` |
| `--текст--` | Удаление (delete) | `--старое--` |
| `/*текст*\` | Комментарий | `/*не видно*\` |
| `$текст$` | Формула (строчная) | `$x+y$` |
| `$$текст$$` | Формула (центр) | `$$x^2$$` |
| `\X` | Экранирование | `\*` → `*` (без форматирования) |

### Правила работы парсера

#### 1. Строгая парность
Форматирование применяется ТОЛЬКО если найдена полная пара «открывающий маркер — закрывающий маркер». Если закрывающий маркер отсутствует — вся конструкция (включая открывающий маркер) считается обычным текстом.

```
**жирный**        → ✅ жирный
**не жирный       → ❌ **не жирный (простой текст)
```

#### 2. Пробелы у маркеров
После открывающего маркера НЕ должно быть пробела. Перед закрывающим маркером НЕ должно быть пробела.

```
**текст**         → ✅
** текст**        → ❌ (пробел после **)
**текст **        → ❌ (пробел перед **)
** текст **       → ❌
```

Пробелы ВНУТРИ контента разрешены:
```
**жирный текст**  → ✅ «жирный текст» целиком жирный
```

Граница строки (начало/конец) считается корректной границей, а не пробелом.

#### 3. Непустой контент
Между открывающим и закрывающим маркером должен быть хотя бы один непробельный символ.

```
****              → ❌ **** (простой текст)
////              → ❌ ////
```

#### 4. Экранирование
Обратный слеш `\` перед любым символом отключает его специальное значение. Сам слеш в отображаемом тексте не показывается.

```
\*\*текст\*\*     → **текст** (как простой текст)
\\                → \
```

#### 5. Приоритет маркеров
Если два маркера начинаются с одних и тех же символов, приоритет у более длинного:

```
$$x$$             → формула по центру
$x$               → строчная формула
!!!текст!!!       → блок спойлер
!!текст!!         → строчный спойлер
```

Маркеры проверяются от самого длинного к самому короткому. совпадение по короткому маркеру происходит только если длинный не подошёл.

#### 6. Вложенность
Маркеры могут быть вложены друг в друга. Стек закрывается строго в обратном порядке (первый открыт — последний закрыт).

```
**жирный //и курсив//**            → ✅
//курсив внутри **жирного**//       → ✅
**a **b** c**                       → ✅ (двойной жирный)
```

Стили комбинируются: вложенный маркер добавляет свой стиль к родительскому.

#### 7. Многострочные маркеры
Некоторые маркеры (`/*`, `$$`, `!!!`) работают через несколько строк. Их закрытие ищется во всём оставшемся тексте, а не только в текущей строке.

```
/*
многострочный
комментарий
*\
```

Обычные маркеры (`**`, `//`, `~~` и т.д.) закрываются ТОЛЬКО в пределах одной строки.

#### 8. Действие для парсера
- Парсер выполняет один проход по тексту (O(n)).
- Разбивает текст на токены: обычный текст, открывающие маркеры, закрывающие маркеры, escape-последовательности.
- Стековый алгоритм проверяет парность и вложенность.
- Результат записывается в существующий кэш (DocumentCache → MarkupCache → Segment).
- Каждый Segment содержит: отображаемый текст, битовую маску стиля, длину открывающих/закрывающих маркеров (для навигации), позиции в сыром и отображаемом тексте.



# Как должен по итогу выглядеть полный парсер

## md+ Symbols

| Символ  | Назначение        | Пример               |
| ------- | ----------------- | -------------------- |
| // //   | Курсив            | //текст//            |
| ** **   | Жирный            | **текст**            |
| __ __   | Подчёркивание     | __текст__            |
| '' ''   | Верхний индекс    | ''текст''            |
| ,, ,,   | Нижний индекс     | ,,текст,,            |
| ~~ ~~   | Зачёркивание      | ~~текст~~            |
| \       | Экранирование     | \#текст              |
| == ==   | Подсветка         | ==текст==            |
| !! !!   | Спойлер           | !!название:текст!!   |
| !!! !!! | Блок спойлер      | !!!название:текст!!! |
| ++ ++   | Вставка           | ++текст++            |
| -- --   | Удаление          | --текст--            |
| %% %%   | Комментарий       | %%текст%%            |
| $ $     | Формула           | $x+y$                |
| $$ $$   | Формула по центру | $$x+y$$              |

---

| Символ | Назначение | Пример |
| --- | --- | --- |
| [] | Определение | [~HTTP:протокол] |

| <> | Параметр | [$ковычки:«<>»] |
| () | Отображение | [[Файл]](текст) |
