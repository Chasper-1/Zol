## zml Symbols (legacy)

| Символ  | Назначение        | Пример               |
| ------- | ----------------- | -------------------- |
| // //   | Курсив            | //текст//            |
| ** **   | Жирный            | **текст**            |
| __ __   | Подчёркивание     | __текст__            |
| '' ''   | Верхний индекс    | ''текст''            |
| ,, ,,   | Нижний индекс     | ,,текст,,            |
| ~~ ~~   | Зачёркивание      | ~~текст~~            |
| \       | Экранирование     | \#текст              |
| == ==   | Подсветка         | ==текст==            |
| !! !!   | Спойлер           | !!название:текст!!   |
| !!! !!! | Блок спойлер      | !!!название:текст!!! |
| ++ ++   | Вставка           | ++текст++            |
| -- --   | Удаление          | --текст--            |
| %% %%   | Комментарий       | %%текст%%            |
| $ $     | Формула           | $x+y$                |
| $$ $$   | Формула по центру | $$x+y$$              |

---

| Символ | Назначение | Пример |
| --- | --- | --- |
| [] | Определение | [~HTTP:протокол] |
| {} | Использование | {~HTTP:текст} |
| <> | Параметр | [$ковычки:«<>»] |
| () | Отображение | [[Файл]](текст) |

---

| Символ | Назначение         | Пример                  |
| ------ | ------------------ | ----------------------- |
| ^      | Сноска             | [^1:текст]              |
| ~      | Термин             | [~название:описание]    |
| !      | Специальный объект | [!Название:текст]       |
| $      | Шаблон строчный    | [$название:«<>»] и т.д. |

---

| Символ  | Назначение          | Пример     |
| ------- | ------------------- | ---------- |
| \#N#    | Заголовок           | \#N# Текст |
| -       | Список              | - элемент  |
| 1.      | Нумерованный список | 1. элемент |
| ` `     | Код                 | `код`      |
| ``` ``` | Блок кода           | ```код```  |
| ---     | Разделитель         | ---        |

---

| Символ | Назначение | Пример |
| --- | --- | --- |
| > | Цитата | > текст |
| > < | Блочная цитата | >текст< |
| \#:текст | Тег | \#:tag |

---

| Символ   | Назначение         | Пример              |
| -------- | ------------------ | ------------------- |
| [[ ]]    | Внутренняя ссылка  | [[Файл]]            |
| [[ ]]( ) | Ссылка с названием | [[Файл]](текст)     |
| [ ]( )   | Внешняя ссылка     | [url](текст)        |
| !< >{ }  | Изображение        | !<width:300>{image} |

---

| Символ | Назначение               | Пример      |
| ------ | ------------------------ | ----------- |
| :---   | Левое выравнивание       |:54          |
| ---:   | Правое выравнивание      | 54         :|
| :---:  | Центральное выравнивание |:54         :|

---

| Символ | Назначение            | Пример           |
| ------ | --------------------- | ---------------- |
| {: }   | Макрос                | {:date}          |
| [$ ]   | Создание шаблона      | [$quote:«<>»]    |
| {$ }   | Использование шаблона | {$quote:текст}   |
| [~ ]   | Создание термина      | [~HTTP:описание] |
| {~ }   | Использование термина | {~HTTP:текст}    |
| [? ]   | Клавиши               | [?:Ctrl+C]       |
